README Summary
1.	Python 3 is installed
2.	Save the source code as .py file
3.	Run the program by using the command python cafe_system.py
4.	In the terminal follow the menu to enter customer name, add items, remove items and generate the bill.
Core Features
•	A text-based UI suitable for future expansion
•	Add, remove and view items within the customer order
•	Automatic bill generation and tax calculations
•	Menu item creation by using Factory Pattern
•	Object oriented design using inheritance and composition
